### Edge cases / limitations phát hiện thêm:

1. **`#[allow(dead_code)]` trên `RegisterIntent`:** Sau khi `CrontabScheduler::register` ship với Linux thực thi `intent.label`, `intent.hour`, `intent.minute`, `intent.self_exe` — nhưng `intent.working_dir` vẫn KHÔNG đọc trong Linux code (Linux crontab line không cần working_dir). Clippy vẫn yêu cầu `#[allow(dead_code)]` trên `RegisterIntent` cho `working_dir` field. Giữ nguyên annotation.

2. **`NoopScheduler` `#[allow(dead_code)]`:** Vẫn cần vì `NoopScheduler` được gated `#[cfg(target_os = "macos")]` tests (qua `cfg(test)` trong `core::*::tests`). Linux clippy vẫn thấy nó là dead code. Giữ annotation.

3. **`cli/unregister.rs` macOS-specific warning messages on Linux:** Khi `was_registered=false` (không có tag line), `UnregisterOutput.plist_existed=false` và `was_loaded=false` → `cli/unregister.rs:44,49` print 2 warning dòng macOS-era (`"launchctl bootout: label may not be loaded..."`, `"plist file already absent..."`). Semantically odd trên Linux nhưng không ảnh hưởng exit code. **Out of scope P013** — watch-item cho P016/P017 khi CLI messages được Linux-aware hóa.

4. **Mock crontab PATH injection pattern:** Không có precedent trong existing tests (P012 Discovery đã ghi). Approach hoạt động tốt — `cmd.env("PATH", format!("{}:{}", mock_dir, existing))` override PATH cho subprocess, không ảnh hưởng test runner process. 7 integration tests pass parallel-safe.

5. **`crontab -` empty content:** Khi `unregister` xóa line duy nhất và `kept.is_empty()`, `write_user_crontab("")` pass empty string. WSL2 behavior: `crontab -` với stdin rỗng → xóa toàn bộ crontab user (equivalent `crontab -r`). Phiếu đã acknowledge. Dogfood smoke xác nhận behavior đúng (before/after diff = 0).

6. **`generate_plist` inline allowlist trong macOS.rs:** Task 2 spec nói "Worker may optionally migrate `generate_plist`'s inline check to `super::is_valid_label`". Worker không migrate — `generate_plist` có inline check `label.chars().all(|c| ...)` (không gọi qua fn) nên không phải duplicate fn, chỉ là duplicate logic. Scope control đúng đắn.
